# java-learning-ground

Download JDK from https://www.oracle.com/java/technologies/javase-downloads.html

## Code > Compile > Package

### Directly run from command line
- java hello.java

### Compile and then run
- javac hello.java
- java hello

###
- javac hello.java
- jar cf hello.jar hello.class
